HowTo

For this example, you need a running broker with a WebSocket STOMP transport and an appropriate message generator. Since bowerick version 2.8.0, the bowerick standalone broker includes an appropriate message generator. Below is an example for starting bowerick broker instance, which includes a matching message generator for this example: 

    java -jar dist/bowerick-2.8.0-standalone.jar -G yin-yang -I 40 -u "tcp://127.0.0.1:1031 mqtt://127.0.0.1:1701 ws://127.0.0.1:1864 stomp://127.0.0.1:2000"

Once the broker is started, you connect to it by pressing the "Connect" button on this page.
